# Performance Requirements (from ROADMAP.md)
- PERF-01: Feed loads initial 10 notebooks in under 2 seconds

- PERF-03: WASM editor initializes in under 5 seconds
- PERF-04: Images are lazy-loaded and optimized

# Performance Budget Thresholds
- API p95 latency < 500ms for feed endpoint
- API p95 latency < 1000ms for compilation endpoint
- API p95 latency < 300ms for search endpoint
- Frontend LCP (Largest Contentful Paint) < 2.5s
- Frontend FID (First Input Delay) < 100ms
- Frontend CLS (Cumulative Layout Shift) < 0.1
- Frontend TTI (Time to Interactive) < 3.5s
