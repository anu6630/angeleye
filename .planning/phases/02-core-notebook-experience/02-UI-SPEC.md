## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|

| sm | 8px | Compact element spacing |
| md | 16px | Default element spacing |

| xl | 32px | Layout gaps |

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | HSL variables (--background) | Background, surfaces, light grays/whites |
| Secondary (30%) | HSL variables (--card) | Cards, sidebar, nav |
| Accent (10%) | HSL variables (--primary) | Primary CTA buttons, active navigation states, focus rings, liked state |
| Destructive | HSL variables (--destructive) | Destructive actions only (delete notebook) |

Accent reserved for: Primary CTA buttons, active navigation states, focus rings on form inputs, and liked notebook state.
