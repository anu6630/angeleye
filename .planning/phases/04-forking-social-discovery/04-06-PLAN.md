    __init__(self, db: Session):
    - Initialize self.db = db
    - Initialize self.client = meilisearch.Client(settings.MEILISEARCH_URL, timeout=settings.MEILISEARCH_TIMEOUT)
    - Initialize self.index_name = settings.MEILISEARCH_INDEX_NAME

    create_index(self) -> None:
    - Try: self.client.create_index(self.index_name, {"primaryKey": "id"})
    - Except meilisearch.errors.MeilisearchError: log warning (index might exist)
    - Configure searchable fields: ["title", "content", "author"]
    - Configure filterable fields: ["parent_id"] (for faceted search)
    - Configure sortable fields: ["created_at"]
    - Set ranking rules: ["words", "typo", "proximity", "attribute", "sort", "exactness"]

    index_notebook(self, notebook: Notebook) -> None:
    - Get all cells: notebook.cells (joined load)
    - Concatenate cell content: content = "\n".join([cell.content for cell in notebook.cells if cell.cell_type == "code"])
    - Build document:
      * id: notebook.id
      * title: notebook.title
      * content: content (all code cells)
      * author: notebook.user.username
      * parent_id: notebook.parent_id (null for originals, int for forks)
      * is_published: notebook.is_published
      * created_at: notebook.created_at.isoformat()
    - Try: self.client.index(self.index_name).update_documents([document])
    - Except: log error, don't fail (sync failures shouldn't break saves)
    - Per CONTEXT.md D-19: Real-time sync on every save, log failures

    search_notebooks(self, query: str, tab: str = "all", limit: int = 50) -> dict:
    - Try Meilisearch first:
      * Build filter: if tab == "originals": "parent_id IS NULL", elif tab == "forks": "parent_id IS NOT NULL", else: None
      * Call self.client.index(self.index_name).search(query, {"limit": limit, "filter": filter})
      * Extract notebook IDs from hits
      * Return {"notebook_ids": [int], "total": int, "from_meilisearch": True}
    - Except meilisearch.errors.MeilisearchCommunicationError:
      * Log warning: "Meilisearch unavailable, falling back to PostgreSQL"
      * Fallback to PostgreSQL:
        - Query: SELECT notebooks.id FROM notebooks JOIN users ON notebooks.user_id = users.id WHERE notebooks.is_published=True AND notebooks.is_archived=False AND (notebooks.title ILIKE '%query%' OR users.username ILIKE '%query%')
        - Apply filter: if tab == "originals": AND notebooks.parent_id IS NULL, elif tab == "forks": AND notebooks.parent_id IS NOT NULL
        - Limit results
      * Return {"notebook_ids": [int], "total": int, "from_meilisearch": False}
    - Per CONTEXT.md D-20: Fallback to PostgreSQL LIKE on connection failure

    bootstrap_index(self) -> None:
    - Check if index exists: self.client.get_index(self.index_name)
    - If exists, return
    - Create index: create_index()
    - Query all published notebooks: SELECT * FROM notebooks WHERE is_published=True AND is_archived=False
    - For each notebook: index_notebook(notebook)
    - Log: "Bootstrapped Meilisearch index with {count} notebooks"

    Example implementation pattern:
    ```python
    def update_notebook(self, notebook_id: int, **kwargs) -> Notebook:
        # ... existing update logic ...
        db.commit()
        db.refresh(notebook)

        # Sync to search index
        try:
            SearchService(self.db).index_notebook(notebook)
        except Exception as e:
            logger.warning(f"Failed to index notebook {notebook_id} in Meilisearch: {e}")

        return notebook
    ```
